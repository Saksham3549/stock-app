# Stock Market Simulator Dashboard

A modern stock market simulation platform built to help users learn investing, track portfolios, and practice trading strategies without risking real money.

**Live Demo:** https://stock-app-navy-zeta.vercel.app/

## About The Project

The Stock Market Simulator Dashboard is designed to provide a realistic investing experience for students, beginners, and anyone interested in understanding how the stock market works.

Instead of using real money, users can explore market trends, manage virtual portfolios, monitor stock performance, and gain confidence in investment decision-making within a risk-free environment.

The goal of this project is to combine financial learning with a modern and intuitive user experience inspired by popular trading platforms such as Zerodha and Groww.
